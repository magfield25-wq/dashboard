KPI 용어정의
상품매출 = 상품 판매금액(최종 상품별 총 주문금액) / 쇼핑몰수수료 = 상품매출 - 정산예정금액
배송비매출 = 고객이 낸 배송비 매출수익 / 내부출고비 = 우리가 지출하는 배송비 / 상품원가 = 단가 × 수량 × 1.1
매출수익 = 상품매출 + 배송비매출
비용 = 상품원가 + 내부출고비 + 쇼핑몰수수료
순이익 = 매출수익 - 비용 / 광고차감 순이익 = 순이익 - 광고비
매출수익 구성 = 쇼핑몰수수료 + 상품원가 + 내부출고비 + 광고비 + 광고차감 순이익

광고비 처리 방식

계산식: 평일일광고비 = 월광고비 ÷ (평일수 + 주말수 × 주말비율), 주말일광고비 = 평일일광고비 × 주말비율
